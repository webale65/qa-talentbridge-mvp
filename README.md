# QA Testing - TalentBridge MVP

Este repositorio contiene la planificación y documentación de QA para el MVP TalentBridge, desarrollado en el contexto de la simulación laboral de No Country.

## Objetivo

Validar la calidad del MVP asegurando el correcto funcionamiento de los flujos principales, la consistencia de los datos y la experiencia del usuario.

## Alcance del Testing

- Autenticación (registro / login)
- Onboarding
- Perfil profesional
- Diagnóstico de habilidades
- Resultados y asignación de nivel
- Ruta de aprendizaje
- Marketplace
- Notificaciones

Nota: El alcance incluye funcionalidades planificadas del MVP. 
La ejecución de testing se realizará de forma progresiva según disponibilidad de desarrollo.

## Enfoque de Testing

- Pruebas funcionales
- Validación de UX
- Testing end-to-end
- Escenarios negativos
- Validaciones de seguridad (datos sensibles)

### Valor QA

Este enfoque permite detectar errores tempranos, reducir riesgos en el flujo principal y asegurar una experiencia clara para el usuario final.
