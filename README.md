# 📋 Descripción

Este repositorio contiene la estrategia de Quality Assurance, planificación de pruebas y documentación QA desarrollada para el proyecto **TalentBridge MVP**, una plataforma orientada a mejorar la empleabilidad mediante validación de habilidades, rutas de aprendizaje personalizadas y conexión entre candidatos y reclutadores.

El trabajo fue realizado durante la simulación profesional de **No Country** aplicando metodologías ágiles y prácticas colaborativas de desarrollo de software.

---

# 👨‍💻 Mi Rol

**QA Tester**

Responsabilidades principales:

* Definición de estrategia QA
* Análisis de riesgos funcionales y de negocio
* Elaboración de planes de prueba
* Diseño de casos de prueba
* Definición de trazabilidad
* Preparación de estructura de evidencias
* Definición de estándares de reporte de bugs
* Participación en procesos ágiles de desarrollo

---

# 🚀 Sobre TalentBridge

TalentBridge es una plataforma diseñada para mejorar la empleabilidad de profesionales mediante:

* Validación de habilidades
* Diagnóstico de competencias
* Rutas de aprendizaje personalizadas
* Marketplace entre candidatos y reclutadores
* Seguimiento de crecimiento profesional

El proyecto fue concebido como un MVP dentro de una simulación profesional enfocada en empleabilidad y validación de habilidades.

# 🎯 Alcance del MVP

El alcance funcional actual contempla:

* Registro y Login
* Onboarding de candidatos
* Onboarding de reclutadores
* Perfil profesional
* Diagnóstico basado en IA
* Validación de habilidades
* Learning Path personalizado
* Dashboard para reclutadores
* Sistema de notificaciones

---

# 🔍 Riesgos Identificados

Algunos riesgos identificados durante la etapa de análisis QA:

* Asignación incorrecta de niveles de habilidad
* Exposición de datos sensibles
* Inconsistencias en filtros del marketplace
* Confusión entre habilidades declaradas y validadas
* Abandono de usuarios durante onboarding o diagnóstico
